⚡️ Mermaid 图表语法高亮预览 `markdown-it` 插件，支持 `SSE` 流式渲染


## 前置条件

* Node >= 20.x
* Pnpm 9.x

## 安装

```shell
npm install @nzoth/markdown-it-mermaid-sse

# or

pnpm add @nzoth/markdown-it-mermaid-sse
```
